Um site interativo e intuitivo que permite aos usuários criar frames personalizados para suas fotos automaticamente a partir dos metadados das imagens. Ao fazer o upload de uma foto, a plataforma lê os metadados embutidos — como data, hora, localização GPS, modelo de câmera, configurações de exposição — e usa essas informações para gerar frames estéticos e contextuais que complementam e enriquecem a imagem.
